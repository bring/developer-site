Revision History

This page contains most important changes and updates for our APIs.

January 10, 2020 - Booking API

Added support for sending booking confirmation e-mail if e-mail is specified in request. Can be applied to sender's e-mail for outgoing services, and both sender's/recipient's e-mail for return services. Note that e-mail is optional and can at any time be skipped from the request.

December 12, 2019 - Reports API

Added InvoiceSource in metadata section of the response when specified invoice report is generated.

December 3, 2019 - Booking API

Decommissioned Courier 6H for Norway from Booking API.

December 3, 2019 - Shipping Guide API

Decommissioned Courier 6H for Norway from Shipping Guide API.

October 31, 2019 - Shipping Guide API

Decommissioned Home Delivery Mailbox from Shipping Guide API.

September 2, 2019 - Event Cast API

Released Event Cast API, an API for getting event updates pushed instead of polling tracking.

August 21, 2019 - Shipping Guide API

Decommissioned Sm√•pakker from Shipping Guide API.

July 8, 2019 - Invoice API

Added new attributes fromDate and toDate to specify period for fetching invoices.

March 22, 2019 - Shipping Guide API

Decommissioned version 9 of the Shipping Guide API. Customers should use Shipping Guide API 2.0 instead.

March 15, 2019 - Shipping Guide API

Decommissioned version 7 of the Shipping Guide API. Customers should use Shipping Guide API 2.0 instead.

March 4, 2019 - Booking API

It is now possible to pass customs declaration information that will be sent electronically to the destination country, using the field ediCustomsDeclarations.

January 31, 2019 - Mailbox Packet API

Add support for bagOnDoor, a delivery alternative for parcels that cannot be delivered to the mailbox.

January 28, 2019 - Booking API

Added new syntax for requesting pickup. Both ways are still supported, but they can't be combined.

January 21, 2019 - Warehousing API

Added new attribute OrderItem/ArticleMonetaryAmounts that contains the article unit sales price.

January 15, 2019 - Booking API

Added new syntax for requesting additional services. Both ways are still supported, but they can't be combined.

November 20, 2018 - Shipping Guide API

Decommissioned version 1 and 2 of the Shipping Guide API. Customers should use Shipping Guide API 2.0 instead.

May 17, 2018 - Modify Delivery API

Released Modify Delivery API, an API used to modify a shipment currently on it's way.

April 9, 2018 - Shipping Guide API

Decommissioned version 10 of the Shipping Guide API. Customers should use Shipping Guide API 2.0 instead.

March 28, 2018 - Shipping Guide API

Decommissioned version 3 of the Shipping Guide API. Customers should use Shipping Guide API 2.0 instead.

March 26, 2018 - Shipping Guide API

Decommissioned version 4, 5 and 6 of the Shipping Guide API. Customers should use Shipping Guide API 2.0 instead.

March 11, 2018 - Warehousing API

The Order Management API split out of the Warehousing API. These will now continue as two separate APIs.
