EDI documentation

If you're an EDI vendor, this is where you will find our implementation guides.

  • Tracking API

    Use a reference, package or shipment number to display details about sender, recipient, weight/volume, shipment history and current status.

  • Shipping Guide API

    Look up available options for shipping with Bring: price information, service area, estimated delivery times, delivery alternatives.

  • Booking API

    A logged-in service for Mybring users that letâ€™s you programatically book shipments and generate PDF labels.

  • Pickup Point API

    List the nearest pickup points. These are locations that deliver packages for Bring and Posten, such as post offices and selected grocery stores.

  • Reports API

    A logged-in service for Mybring users. Generate shipment reports, the same reports that are available for you about a set of customers in Mybring.

  • Purchase Order API

    Create, update and query Purchase Orders.

  • Easy Return Service API

    A logged-in service for Mybring users to create labels for returning cross‑border parcels.

  • Postal Code API

    Check if a number is a valid postal code, and look up name of the geografic area. For Norway and some additional supported countries.
