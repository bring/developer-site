Dangerous goods details

When ordering dangerous goods, some additional information describing the goods is required. The following elements/attributes must be sent if they are defined for the specific UnNumber:

  • unNumber (mandatory)
  • containerId (mandatory when ordered with OIL_EXPRESS)
  • weightInGrams (either this or)
  • volumeInLitres
  • commodity (mandatory if specified)
  • packingGroup (mandatory if specified)
  • dangerClass (mandatory if specified)
  • safetyMarkCode (mandatory if specified)
  • tunnelRestrictionCode (mandatory if specified)
