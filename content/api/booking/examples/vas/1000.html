Payment on delivery

The following element/attributes are available with the additionalService instance when ordering Cash On Delivery:

  • accountNumber (mandatory)
  • accountType (mandatory if currencyCode = 'SEK')
  • amount (mandatory) Accepts numeric values with two decimal places, with a dot (not comma) as decimal separator
  • currencyCode (mandatory)
  • codMessage (optional)

Possible accountType values:

  • bankgiro
  • plusgiro

Possible codMessage.type values:

  • kid
  • text
