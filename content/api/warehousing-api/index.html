Integration Options:

At Bring Shelfless, we offer three well-established integration methods to connect with our logistics and fulfillment platform:

  1. New Shelfless API

    Recommended for New Integrations

    Our modern REST API platform is designed for faster implementation, improved scalability, and ease of use. It follows standard RESTful principles with JSON formatting, making it developer-friendly and ready for future enhancements. https://www.api-doc.shelfless.io/

  2. Plug ´n play integration

    For common e-com platforms, such as Shopify, Woocommerce, Magento2 among others, we have plug ´n play integrations. Please hit the "Contact us" button on this link if you want to connect your online store with us https://www.bring.no/en/services/warehousing/shelfless/

  3. Legacy Integration

    A stable and proven solution, the legacy integration is used successfully by many of our customers today. It continues to be supported. This option is ideal for existing partners or systems already built around this integration model. https://developer.bring.com/api/warehousing//

Choosing the Right Integration

  • All integrations are reliable and supported
  • We recommend the Shelfless API for all new ERP integrations due to its modern design and extended capabilities
  • If you want to connect you e-com platform with Bring Shelfless we recommend our plug ´n play solution
  • The legacy integration remains a solid and supported option, especially for partners with established systems
