If you are looking for an alternative to directly integrating with our APIs, Posten Bring provide ready-made plugins for popular shopping platforms like WooCommerce and Shopify. These plugins allow you to quickly enable our shipping services in your webshop without custom development.

Available Plugins:

  • WooCommerce Plugin
  • Shopify Plugin

When to Use Plugins:

Use these plugins if you want a simple setup and prefer not to develop custom integrations with our APIs. They are ideal for smaller shops or those using standard platform features that align with plugin functionality.

Direct API Integration vs. Plugins:

Direct API Integration Plugins
Customizable and tailored to your needs Quick and easy setup
Requires development effort No coding required
Access to all API features Limited to plugin-supported features

Getting Started:

To use the plugins, you will need an account in MyBring. Follow these steps to get started:

  1. Create or log in to your MyBring account at mybring.com.
  2. Install the plugin in your webshop.
  3. Register your webshop at mybring.com/checkout.
  4. Activate the plugin by entering your MyBring credentials in the plugin settings within your webshop.

Once these steps are completed, your webshop will be ready to use Posten Bring's shipping services.
