Shipping Guide API 1.0 will finally be decomissoned on 01.07.2021

3 years ago, Shipping Guide API 2.0 was launched. Since then, schemas belonging to 1.0 has been phased out one by one. The last active schema, 11, will be decomissioned on 01.07.2021. If you haven't upgraded to Shipping Guide 2.0 yet, we strongly recommend to do that as soon as possible, and latest by 01.07.2021.
