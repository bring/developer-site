Param Type Description
pickupPoint Array List of pickup points. The array will be empty if there is no match on the postal code.
id String Pickup point ID
unitId String
name String The name of the pickup point
address String The address of the pickup point
postalCode String The postal code of the pickup point
city String The city of the pickup point
countryCode String The country code of the pickup point
municipality String The municipality of the pickup point
county String The county of the pickup point
visitingAddress String The visiting address of the pickup point
visitingPostalCode String The visiting address postal code of the pickup point
visitingCity String The visiting address city of the pickup point
openingHoursNorwegian String Opening hours of the pickup point in Norwegian language
Example: Man-Fre: 0800-2200, Lør: 0800-2000
openingHoursEnglish String Opening hours of the pickup point in English language
Example: Mon-Fri: 0800-2200, Sat: 0800-2000
openingHoursFinnish String Opening hours of the pickup point in Finnish language
Example: Maa-Per: 0800-2200, Lau: 0800-2000
openingHoursDanish String Opening hours of the pickup point in Danish language
Example: Man-Fre: 0800-2200, Lør: 0800-2000
openingHoursSwedish String Opening hours of the pickup point in Swedish language
Example: Mån-Fre: 0800-2200, Lör: 0800-2000
latitude Double Map coordinates
longitude Double Map coordinates
utmX String Map coordinates in the utm format
utmY String Map coordinates in the utm format
postenMapsLink String Link to pick up point in Posten Maps
googleMapsLink String Link to pick up point in Google Maps
distanceInKm String The distance in km to pickup point from current location
distanceType String The type of distance, can be Driving Distance if match is found by Google Maps. If no match is found the overhead line is used instead.
type String

The type of Pick up point

Norway:

  • Type 4: Post i Butikk
  • Type 19: Posten pakkeutlevering
  • Type 21: Postkontor
  • Type 27: Pakkeautomat
  • Type 32: Postpunkt, egendrevet
  • Type 33: Postpunkt
  • Type 37: Pakkeboks

Sweden:

  • Type 34: Manned

Denmark:

  • Type 85: Manned
  • Type 86: Locker

Finland:

  • Type SmartPOST: Finland Smart Post Pickup
  • Type Posti: Finland Posti Pickup Point
  • Type Noutopiste: Finland Pickup Point
  • Type LOCKER: Finland Locker Pickup Point
additionalServiceCode String
routeMapsLink String Link to pick up point in Bring Maps
