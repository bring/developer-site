--- title: List available customer numbers layout: api notanapi: true --- 
  GET
    https://api.bring.com/booking/api/customers.json
https://api.bring.com/booking/api/customers.xml

Success response

200

This can mean that a required field or parameter has not been provided, the value supplied is invalid, or the combination of provided fields is invalid.

Param Type Description
customers Array A list of your customer numbers and the services attached to it
name String The name of the customer account
customerNumber String The customer UUID, prefixed with the name of the specialist e.g PARCELS_NORWAY, followed by the number
countryCode String Countrycode in ISO 3166-1 alfa-2 format
products Array A list of the services that are available to this customer number

Error response

Unauthorized (401)

A failed response when the request is unauthenticated
