In addition to [authentication](/api/#authentication), you need to be authorized with the _booking_ right in order to submit customs data. This is done in [customer administration in Mybring](https://www.mybring.com/useradmin-external/users):

![](/images/authorization.png)

To send customs data, you must specify which customer number to use.
